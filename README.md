中学受験：わが家の「脳内環境」3分診断ツール
このツールは、中学受験という過酷な環境下で、お子さんの「知能を最大化する家庭環境」を可視化するための教育脳科学アセスメントツールです 。

🧠 開発の背景：なぜ「脳内環境」なのか？
中学受験における最大の敵は、知識不足ではなく、ストレスによる「脳のフリーズ（扁桃体の過覚醒）」です 。
デビッド・ロック博士が提唱した「SCARFモデル」に基づき、家庭内における「脅威」を取り除き、「報酬」を最大化することで、お子さんのIQとレジリエンスを引き出すことを目的としています 。

🛠 診断の5つの柱（SCARFモデル）
本ツールでは、以下の5つの観点から家庭の脳内環境をスコアリングします。

Status（地位・自己肯定感）： お子さんが一人の人間として尊重され、有能感を感じられているか 。

Certainty（確実性・見通し）： 学習スケジュールや目標が明確で、脳が「安心」して集中できているか 。

Autonomy（自律性・自己決定）： 「やらされる勉強」ではなく、自分の意志で選択している感覚があるか 。

Relatedness（関係性・信頼）： 親子が評価を超えた強い絆で結ばれ、家庭が「安全基地」になっているか 。


Fairness（公平性・納得感）： 努力が正当に認められ、家庭内のルールに納得感があるか 。

🚀 使い方
診断の実施： 15問の質問に、直感で回答してください。

スコアの確認： レーダーチャートで、わが家の「強み」と「伸び代」を視覚的に把握します。

魂の処方箋： 各カテゴリのスコアに応じた、具体的な脳科学アドバイス（魂のフィードバック文）を確認します。

連載記事との連動： 課題が見つかった項目については、対応する『教育脳科学note』の記事で深い対策を学びます 。
